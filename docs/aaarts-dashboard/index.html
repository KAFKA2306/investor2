次の進化

失敗を削除せず、次の探索空間を狭める制約として保存する。

今回得た知識

  • 古く著名な仮説でも、後半期間まで一貫して残るとは限らない。
  • 早期ホールドアウトだけを採用根拠にすると、時間的不安定性を隠す。
  • 小さなコスト控除でもリスク調整後成績は急速に弱くなる。
  • 負の成果物を保存すること自体が、重複探索の抑制になる。

PIVOT条件

  • 2018年以降を独立に凍結し、既存結果を見ずに再検証する。
  • 個別銘柄・PIT・取引可能性を含む再構築へ移行する。
  • 別市場または別メカニズムで、同じ失敗理由を回避できる候補を探索する。
  • 新候補は、この候補との重複と既知因子依存を明示する。
