AI探索の検証ロジック

AIの役割は仮説実装・大量検証・反証補助。探索履歴や市場制度を省略する権限は与えない。

GO条件

  • 完全な試行集合とパラメータ空間を先に固定する。
  • 選択期間と最終テスト期間を時間順に完全分離する。
  • 多重検定補正後も統計的ハードルを通過する。
  • Point-in-Time銘柄集合・上場廃止・企業行動を再構築する。
  • シグナル時点での売買可能性、空売り可否、借株費、価格制限を適用する。
  • スプレッド、回転率、インパクト、税を含めても正の期待値が残る。
