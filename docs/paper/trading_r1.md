# さいきょうの推論マシン：Trading-R1 である

**代表的な論文**: *Trading-R1: Financial Trading with LLM Reasoning via Reinforcement Learning*  
**発表年**: 2025年  
**お悩み解決の方針**: 「AIによる購入根拠は何か」「根拠はどこにあるのか」といった懸念を解消するため、判断過程を論理的なステップで全て解明する。

## エグゼクティブサマリー
Trading-R1 は、OpenAI の o1 に類似し、じっくり考えてから動くことが可能な、次世代の強化学習（RL）トレーダーである。基礎から応用まで訓練を重ね、推論プロセスを蒸留（Distillation）により模倣させることで、人間に匹敵する深い洞察力を獲得した。GRPO という最適化手法を用い、論理の正当性と利益の両立を追求する。本モデルは高い推論能力を有し、投資判断の品質向上を目指す。

## 推論アプローチの変化
強化学習（RL）で培った推論能力を用いて、個別の投資判断を丁寧に読み解く。本モデルは単なる予測にとどまらず、市場の背後にある論理を重視するものである。

## 本モデルの特徴 3 点
1. Three-Stage Curriculum（三段階カリキュラム）: 形を学習する段階、データで裏付ける段階、利益を最大化する段階、という三段階プロセスで、最適な推論能力を育成する。  
2. Reverse Reasoning Distillation（推論過程の模倣）: 賢いモデルの思考過程を観察・模倣することにより、深い推論プロセスを習得する。  
3. GRPO Strategy Optimization（戦略最適化）: 論理の一貫性とリスクを考慮したリターンを追求する。

## Gen 4 への効果
- 深い納得感: 推論過程の痕跡が明示されるため、提案の信頼性が高まり、協働を促進する。  
- 複雑な市場状況にも対応可能: 簡単なルールだけでは対応できない変動にも適切に理解・対応できる。
