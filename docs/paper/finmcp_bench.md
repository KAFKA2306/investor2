# FinMCP-Bench: 実世界の金融ツール活用能力の評価

**タイトル**: FinMCP-Bench: Model Context Protocol に基づく実世界の金融ツール利用のための LLM エージェントのベンチマーク  
**お仕事の目的**: AIエージェントが金融ツール（MCPツール）をどの程度適切に活用できるかを、世界的に厳密かつ公正に評価することを目的とする。  
**解決したいお悩み**: 金融の高度な道具の適切な使用がAIエージェントによって実現されているかを検証し、運用上の課題を解消すること。

## エグゼクティブサマリー
2025年10月（arXiv:2510.05122）に公開された、最新の道具使用テストである。MCP（Model Context Protocol）を用い、プロフェッショナルレベルの実行力を厳格に評価する。613件の実世界サンプルと600件以上の実践課題を通じて、AIエージェントが単に知識を有するだけでなく、実際に動作させる能力を備えたエージェントへと成長させることを目指す。

## 考え方の革新
単に口頭で回答するだけではなく、65種類以上の実世界の金融MCPツールを実際に操作し、ポートフォリオの作成やリスク評価などを行い、実行可能性を徹底的に検証する。

## ここが特長となる3つの要点
1. 613件の実世界サンプル: 実務現場で生じる多様な課題を含む、実践的なテストを収集している。600件以上の実践課題も併せて扱う。  
2. MCPネイティブ評価: 最新のMCP規格に完全対応。AIホストとサーバ間の通信の円滑性を評価する。  
3. 複数ツール連携: 単一ツールだけでなく、複数の道具を組み合わせて問題を解決する能力、すなわち段取りの良さを評価する。

## Gen4 の利用者への効果
- 実務に直結する能力: 理解のみならず、実際に動かせるエージェントの育成を支援する教材となる。  
- さまざまな金融データソースとの連携: MCPにより安定した接続を確保し、データソースとの協調性が高いエージェントの実現を促す。
