# さいきょうのまとめ：FactorMiner

**タイトル**: *FactorMiner: A Self-Evolving Agent with Skills and Experience Memory for Financial Alpha Discovery*  
**お仕事の目的**: 本研究は、自己主導によるアルファ（収益源）の発見を促進する能力を高め、自己進化を可能とすることを目的とする。  
**解決したいお悩み**: 過去の失敗に関する記憶が不足するため、同じ誤りを繰り返す事象を是正する。

## エグゼクティブサマリー
FactorMiner は、失敗を「経験」として記憶する機構である **Symbolic Experience Memory** と、推論と実行を分離する設計思想である **Skill-Module Decoupling** を活用して、自律的なアップデートを実現するエージェントである。これにより、過去に誤って踏み込んだ領域（Forbidden Area）へ再び侵入することを回避し、最適なアルファを迅速に発見できるようになる。経験の蓄積とともに、性能は向上する。

## 考え方の要点
本研究は、アルファを発見する手法と、それを自己回顧によって言語化・記憶化する **Ralph Loop** を導入し、自己改善能力を高める。失敗時には次善案を反省・抽出することを可能とし、将来の意思決定に資する。

## ここが重要な3点

### 1. Symbolic Experience Memory
失敗を単なる数値データとしてではなく、「この理由で不適切だった」という理由づけを伴う言語的表現として記憶する。これにより、同じミスを繰り返さず、経験豊富な判断を形成する。

### 2. Skill-Module Decoupling
推論と実行の役割を分離する設計により、推論ミスを分離・低減し、計算の正確性と信頼性を向上させる。

### 3. Forbidden Area Mapping
利益を生まない領域を明確に識別して地図化し、寄り道を排除してアルファの探索を効率化する。

---

## Gen 4 への効果
- 過去の失敗を活用して自己改善を継続的に進め、性能を向上させる。  
- 計算エンジンの信頼性が確保されており、検出されたアルファはそのまま実行可能である。  
- 自己進化は継続的であり、外部指導を受けずとも自己改善を継続できる。将来の市場において有効に機能する可能性が高い。
