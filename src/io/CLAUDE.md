# src/io/ CLAUDE.md

本資料は外部システムとのデータ交換を担当するモジュールである。データの取得を主眼とし、内部の難解な処理には関与しない。

---

## 関心の分離 (Responsibility)

- **役割**: APIおよびDBから生データを取得・保存すること。
- **含まれるもの**: `get.ts`（API同期）、クライアント定義など。

## 禁止事項 (Strict Rules)

- **加工の禁止**: データを整える前処理ロジックを本モジュールに実装してはならない。前処理は preprocess/ の担当である。
- **依存ルール**: `src/shared` のみをインポートしてよい。

## 検証手順 (Verification)

- `.env` に APIキーが設定されていることを確認する。
- `bun run src/io/get.ts` を実行して、生データがキャッシュされることを確認する。
