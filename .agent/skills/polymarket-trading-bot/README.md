# Polymarket Trading Bot

Autonomous prediction market trading bot using a 6-agent swarm architecture.

## Features

- Scans Polymarket for active markets
- Predicts market outcomes with XGBoost + LLM analysis
- Risk validation using Kelly Criterion and Value-at-Risk (VaR)
- Generates backtest signals for 90-day rolling windows
- Edge detection and learned failure pattern analysis
